# 第四十三章 分班与“吃人”的试卷

“丁级班？”
萧千卷拿着入学通知书，看着上面鲜红的“丁”字，陷入了沉思。

白鹿书院分甲乙丙丁四级。
甲级是天之骄子，预备役进士。
乙级是中规中矩，希望能考个举人。
丙级是混日子的富二代，也就是赞助费给得多的。
至于丁级……

“那是‘废材集中营’。”
负责带路的书童（刚才那个护卫武僧变的）一脸鄙夷。
“也就是所谓的……无可救药之徒。”

萧千卷和墨染被带到了一间位于书院最角落、甚至还能闻到隔壁猪圈味道的破旧教室前。
“到了。这是你们的教室。里面的学生……都很‘活泼’。”
书童说完，像逃命一样跑了。

还没进门，就听到里面传来一阵喧闹声。
“这首诗不通！平仄不对！”
“去你大爷的平仄！老子这叫‘古风’！懂不懂？”
“打他！用砚台砸他！”

萧千卷推开门。
啪。
一块飞来的砚台被他随手接住。
教室内瞬间安静下来。
三十几双眼睛齐刷刷地盯着他。
这些学生，有的在睡觉，有的在斗蛐蛐，有的在画春宫图，唯一的共同点就是……没一个在读书。

“你是谁？”一个满脸横肉的学生（丁级班老大·项大力）站了起来。
“新来的插班生。”
萧千卷把砚台放在讲台上，顺手拿起一支粉笔。
“另外，今天是摸底考试。”

“考试？”项大力笑了，“我们丁级班从来不考试。”
“那是以前。”
萧千卷指了指黑板。
“现在，我来了。规矩就得变。”

他在黑板上写下两个大字：【礼教】。
“今天的题目：何为礼？”

“切！老掉牙的题目。”
“无聊。”
学生们起哄。

萧千卷微微一笑。
“别急。我的要求是……说真话。”
他眼神一凛。
“谁写得好，我教他……怎么把圣人像骂哭。”

这话一出，全班再次安静。
这新来的……有点东西啊？

一炷香后。
萧千卷收卷。
大部分人写的都是“礼者，理也”之类的废话。
唯独萧千卷自己，提笔在最后一张白纸上，写下了一行字。
这是他给自己的“投名状”，也是给那个一直在暗中观察的院长看的。

题目：【狂人日记·礼教】
内容：
“我翻开历史一查，这历史没有年代，歪歪斜斜的每页上都写着‘仁义道德’几个字。我横竖睡不着，仔细看了半夜，才从字缝里看出字来，满本都写着两个字是‘吃人’！”

写完，他将试卷贴在了教室后墙最显眼的位置。
墨染看着那行字，只觉得一股寒意从脚底升起。
“你疯了？在儒家书院写这个？”
“不疯魔，不成活。”萧千卷眼中闪烁着异样的光芒，“这叫……【文道·反讽】。”

（第四十三章·完）
