# 第四十四章 丁字班的“规矩”

“文道·反讽”
这个技能不仅能写文章，还能……拉仇恨。

自从昨天萧千卷在黑板上写下那段话，并且“教导”项大力写了一篇《论夫子之更年期症状》的奇文后，丁级班似乎被激活了某种奇怪的属性。

清晨。
萧千卷踏入教室，发现气氛不对。
原本乱糟糟的教室，此刻出奇的安静。
所有人都坐在座位上，但眼神都盯着讲台。

讲台上，放着一个水桶。
门框上，夹着一块黑板擦。
地上，撒了一层黄豆。

“呵，小儿科。”
萧千卷站在门口，并没有进去。
他只是轻轻打了个响指。
“旺财。”

“吼！”
一声虎啸（虽然听起来像猫叫但分贝足够大），一只白色的影子窜了进去。
噼里啪啦。
水桶扣在了旺财头上。
黑板擦砸在了它背上。
黄豆……被它踩成了粉末。

旺财甩了甩身上的水，幽怨地看着萧千卷。
“喵呜……”（你是魔鬼吗？）

萧千卷淡定地跨过门槛。
“感谢我的宠物替我洗了个澡。现在，上课。”

“我不服！”
项大力猛地拍桌子站起来。
“你凭什么管我们？就凭你会写几句骂人的话？”
他在班里是武力担当，据说是兵家旁听生，一身横练功夫。

“凭这个。”
萧千卷举起右拳。
墨染在后面默默捂住了眼睛。

“你想打架？”项大力笑了，捏得指关节咔咔作响，“我可是练过‘铁布衫’的……”

“不是打架。是讲道理。”
萧千卷突然发动【训诂】。
“定义：你的铁布衫=纸糊的。”
砰！
他一拳轰出。
看似平平无奇的一拳，却带着某种规则层面的穿透力。
项大力这辈子没见过这种打法。
他引以为傲的护体真气像是遇到了克星，瞬间瓦解。
整个人像个沙袋一样飞了出去，挂在了后墙的黑板上，正好遮住了那个“吃人”的字。

全场鸦雀无声。
这特么是读书人？

萧千卷吹了吹拳头。
“还有谁想讲道理？”

角落里，一个一直在玩铜钱的瘦弱少年（阴阳家·神棍）突然怪笑起来。
“有意思。我在你身上算不到命运的轨迹……你是……变数？”
“不算命，算算这个。”
萧千卷扔给他一道数学题。
“鸡兔同笼，头三十五，脚九十四，问鸡兔各几何？”

神棍愣住了。
他手指疯狂掐算，铜钱撒了一地。
“这……这是数术？不对……这是天机！”
他的大脑显然处理不了这种方程组问题。

萧千卷走上讲台，依然是那副懒散的样子。
“文能提笔安天下，武能上马定乾坤。”
“想学吗？”
“想学……就先把那身丑爆了的校服脱了。”

（第四十四章·完）
