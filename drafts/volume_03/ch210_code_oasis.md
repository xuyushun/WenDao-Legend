# 第二百一十章 寻找水源：代码绿洲

白色的寒气逐渐散去。

玄武号的警报声终于停了，老沙头顶的显示器也重新亮了起来，虽然还有几个坏点，但至少不再闪烁雪花。

“这下面有个空间。”老沙将它的长手臂伸进那个被激光轰开的破洞里探测了一番，“深度大约五十米，而且……我检测到了高浓度的活性数据反应。”

“活性数据？”墨染一边喝着刚刚冷凝下来的水，一边问，“是说那里有网？”

“不止是网。”老沙的声音带着一丝兴奋，“那波形……像是某种温室生态系统。”

三人商量了一下，决定下去看看。毕竟光靠这喷出来的液氮只能解一时之急，想要穿越剩下的沙漠，必须找到真正的补给补。

他们顺着热导管内侧的维修梯爬了下去。
越往下，温度越适宜。那种能够烤干灵魂的燥热彻底消失了，空气中甚至弥漫着一股淡淡的清香——那是由于高纯度冷却液挥发而产生的特有味道，但在此时的他们闻来，简直就是仙气。

终于，脚踏实地。
当他们看清眼前的景象时，都不由自主地屏住了呼吸。

这是一个巨大的地下空洞。
头顶并不是漆黑的岩石，而是一块巨大的、模拟天空的穹顶屏幕，虽然大半部分已经坏了，但在剩下的区域里，依然投射着柔和的星光。

在星光之下，是一片奇异的“丛林”。

这里的树木不是木头做的，它们有着半透明的光纤树干，枝叶是无数闪烁着绿色荧光的电路板。
地上的草丛是柔软的数据线缆，随着微风（通风系统）轻轻摇摆。
而在丛林中央，有一潭碧绿的湖水。那湖水不是普通的液体，而是高浓度的“液态文气”——也就是被液化的原始代码。

“我的天……”墨染伸手摸了一下一朵盛开的“晶体花”，“这花瓣居然是有触感的？”

“这是‘代码绿洲’。”萧千卷深深吸了一口气。体内的文气在欢呼，在雀跃，刚才枯竭的经脉几乎在瞬间就被这浓郁的环境填满了。

“上古时代，程序员们为了在枯燥的维护工作中寻找乐趣，利用服务器的剩余算力，在底层构建了这样的虚拟花园。”萧千卷走到湖边，掬起一捧发光的湖水，“这里的数据没有被加密，是最纯粹的开源代码。”

“能喝吗？”墨染问。

“对身体来说，这就是最好的能量饮料。”萧千卷喝了一口。
一股清凉的感觉瞬间流遍全身，大脑变得前所未有的清晰。

“不仅能喝，还能‘醉’人。”

他指着湖边的一排架子。
架子上爬满了紫色的藤蔓，藤蔓上挂着一串串晶莹剔透的……葡萄？

不，那是一个个封装好的数据胶囊，外形酷似葡萄。

“这是‘夜光杯’葡萄。”老沙扫描了一下，“实际上是古代诗人留下的‘灵感碎片’。每一个胶囊里，都封存着一段绝妙的诗句或是一段未完成的旋律。”

萧千卷摘下一颗“葡萄”，轻轻捏碎。
一道紫色的光芒逸散出来，在空中化作一行行飘逸的草书：

*葡萄美酒夜光杯，欲饮琵琶马上催。*

伴随着文字，还有隐约的琵琶声在空旷的地下回荡，带着一丝苍凉，又带着一丝狂放。

“是王翰的《凉州词》。”萧千卷沉醉地闭上眼，“这不仅是诗，更是一段被封存的‘边塞意境’。在这里吃下它，就能获得那种虽千万人吾往矣的豪情与力量。”

“好东西！”墨染眼睛一亮，也摘了一颗塞进嘴里，“看看这个是什么……哇！好辣！”

她吐着舌头，空中浮现出一句红色的诗句：
*男儿何不带吴钩，收取关山五十州。*

“这是李贺的诗，带杀气的。”萧千卷笑着解释，“看来这个绿洲里，种的都是‘战斗系’的作物。”

“那正好。”墨染擦了擦嘴角的“汁水”，“有了这些东西，我们不仅能补满状态，以后要是遇到那些不长眼的家伙，我还能喷他们一脸‘唐诗三百首’。”

老沙则在湖边的终端机上忙碌着：“我也找到了好东西。这里有一个隐藏的‘机房维护日志’。虽然大部分损坏了，但我找到了这片沙漠的地下水脉图。有了它，我们就不用在上面像无头苍蝇一样乱撞了。”

在这个与世隔绝的地下绿洲里，三人难得地放松了下来。
头顶是模拟的星空，身边是流淌的代码之泉，嘴里品尝着来自盛唐的豪情。

“休息一晚吧。”萧千卷靠在那棵发光的大树下，看着手中那杯用冷却液盖子盛满的“葡萄美酒”。

“明天，还要继续西征。”

（第二百一十章·完）
