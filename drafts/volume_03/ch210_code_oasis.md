# 第二百一十章 寻找水源：代码绿洲

白色的液氮寒气逐渐在炎热的空气中消散，留下一地的冰霜痕迹，转眼间又被高温蒸发殆尽。

玄武号那如同哮喘病人般的引擎轰鸣声终于平息了下来，取而代之的是一种令人不安的死寂。车厢内的警报灯虽然不再闪烁，但那红色的光芒依旧刺眼，提示着各项指标都已到达临界点。

“水循环系统效率……12%。”老沙瘫在副驾驶座上，头顶的显示器时而雪花，时而黑屏，电子音听起来像是一盘被磁化了的旧卡带，“我的核心处理器……现在能在他娘的铁板上煎鸡蛋……不，煎牛排都行……”

墨染趴在方向盘上，连翻白眼的力气都没了。她那一头标志性的短发此刻湿漉漉地贴在头皮上，汗水顺着苍白的脸颊滑落，滴在龟裂的皮质座椅上。

“少废话……”她虚弱地骂一句，“再吵……把你拆了……当风扇用……”

萧千卷坐在后座，怀里死死抱着那本泛黄的《全唐诗》。他的嘴唇干裂起皮，儒衫早已被汗水浸透，又被风干，结出一层白色的盐霜。

“‘大漠孤烟直，长河落日圆’……”他双眼无神地盯着车顶，喃喃自语，“古人……写诗的时候……一定没遇上……这么要命的……沙尘暴……”

“老萧……你能不能……念点……带水的……”墨染感觉喉咙里像是有把刀在割。

“‘君不见……黄河之水天上来’……”

“没有……只有沙子……”

三人陷入了沉默。
在这片被称为“绝地”的荒漠中，断水就意味着宣判死刑。玄武号虽然是量产型战车中的佼佼者，改装后更是性能卓越，但也扛不住这连续三天的高强度越野和极端高温。

就在这时，老沙的一只机械臂突然抽搐了一下，随后猛地伸长，钻进了之前被墨染切开的那个热导管破洞里。

“嗯？”老沙的显示器突然亮了一下，那是一个惊叹号。

“咋了……那个洞里……有鬼啊……”墨染有气无力地问。

“这下面……有个空间。”老沙的声音突然变得清晰起来，带着一丝难以置信的颤抖，“深度大约五十米，而且……我检测到了活性数据反应！极高浓度的活性数据！”

“活性数据？”萧千卷猛地坐直了身子，眼中闪过一丝精光，“你是说……下面有网？”

在这个废土世界，无论是生物还是机械，本质上都是数据的载体。对于他们这种修炼“文道”或者改造过的人来说，“高纯度数据”往往意味着能量，意味着生存。

“不止是网。”老沙将更多的数据探针深入地下，“那波形……非常稳定，非常有规律……就像……就像是一个正在运行的生态系统！”

墨染和萧千卷对视一眼。
“挖！”
两人异口同声。

……

半小时后。
依靠玄武号仅存的一点电量驱动的激光切割器，他们在那个破损的热导管下方切开了一个足以容纳一人通过的入口。

一股清凉的气流瞬间从洞口涌出。
那不仅仅是温度上的凉爽，更是一种夹杂着薄荷、臭氧和某种古老纸张霉味的独特气息。

“我先下。”墨染二话不说，抓起缆绳就滑了下去。作为机械师，她对各种地下结构有着天然的敏感。

萧千卷紧随其后，老沙则把自己折叠成一个铁球，“咕噜噜”地滚了下去——这是它最省电的移动方式。

五米，十米，二十米……
随着深度的增加，周围的温度急剧下降。原本那种能够烤干灵魂的燥热彻底消失了，取而代之的是一种令人毛孔舒张的舒适感。

终于，脚踏实地。
墨染打开了肩灯。
当光柱划破黑暗，照亮眼前的景象时，三人都呆立当场，连呼吸都忘记了。

这是一个巨大的地下空洞，大得足以塞进半个长安城的坊市。
头顶并不是漆黑压抑的岩层，而是一块巨大的、呈拱形的穹顶屏幕。虽然这块屏幕的大部分区域已经因为年代久远而坏死，呈现出死寂的黑色，但在剩下的那三分之一区域里，依然顽强地投射着柔和而璀璨的星光。

那是大崩溃之前的星空，纯净，深邃，没有辐射云的遮挡。

而在这一片虚假的星空之下，是一片奇异到令人窒息的“丛林”。

“我的天……”墨染喃喃自语。

这里的“树木”，并不是由木质纤维构成的。它们有着半透明的、如同水晶般的树干，那其实是某种高强度的光导纤维束。树干内部流淌着淡淡的荧光，那是正在传输的数据流。
树冠则是由无数块精密的绿色电路板层叠而成，像是一片片巨大的翡翠叶子。

地上的“草丛”，是数不清的柔性数据线缆，红黄蓝绿交织在一起，随着不知从何处吹来的微风——或许是地下通风系统的气流——轻轻摇摆，由于线缆之间偶尔的接触，不时迸发出微弱的电火花，就像是草丛中飞舞的萤火虫。

而在丛林的中央，有一潭碧绿的湖水。
那平静的湖面像是一块巨大的绿松石，倒映着头顶的星空和周围的赛博森林。

“水……是水！”
墨染的理智在这一刻断线了。她像是看见肉骨头的饿狗，发疯一样冲向湖边。

“等等！小心有毒！”老沙在后面喊，但它自己滚动的速度比谁都快。

墨染冲到湖边，一个急刹车，扑通一声跪倒在岸边。
她伸出手，想要掬起一捧清水。

然而，当她的手指触碰到湖面的瞬间，并没有激起水花，而是荡漾开一圈圈由无数个“0”和“1”组成的涟漪。

“这……”墨染愣住了。
手中的触感冰凉、顺滑，有着液体的流动性，但并没有那种湿润感。

“这不是普通的水（H₂O）。”
萧千卷缓缓走了过来，他的目光中充满了震撼，甚至带着一丝朝圣般的虔诚，“这是高浓度的‘液态文气’……也就是被液化的原始代码（Source Code）。”

“上古时代，也就是大崩溃之前的黄金纪元，程序员们——那时的‘墨家工匠’——为了在枯燥的代码维护工作中寻找乐趣，或者是为了给后人留下文明的火种，利用深层服务器的剩余算力，在底层构建了这样的虚拟花园。”

萧千卷蹲下身，长袖拂过湖面。
“这里的每一个字节，都没有被加密。这是最纯粹的、最原始的开源精神。”

“废话少说，能不能喝？能不能救命？”墨染急得眼睛都红了，“老娘快渴死了！”

“对普通人来说，这是剧毒的数据洪流，喝一口就会脑死亡。”萧千卷转过头，看着墨染，嘴角露出一丝微笑，“但对于我们——对于修习文道，或者进行了机械化改造的人来说，这就是这世上最好的能量饮料。”

说完，他率先低下头，像个豪饮的侠客，将脸埋进制冷的“湖水”中，大口吞咽。

墨染见状，再也不犹豫。
她双手捧起那一汪碧绿的流光，送到嘴边。

入口。
冰凉。
那不仅仅是物理上的低温，更是一种直透灵魂的清冽。那液体顺着喉咙滑下，瞬间化作无数股温和而强大的能量流，钻进了四肢百骸。

墨染能清晰地感觉到，自己体内那些因为高温而几乎停摆的纳米机器人，在接触到这股能量的瞬间，像是打了鸡血一样复活了。它们欢呼着，雀跃着，顺着血管急速奔涌，修复着受损的肌肉纤维，滋润着干枯的神经末梢。

“啊……”
墨染长长地吐出一口气。这一口气吐出，仿佛将这一路上的疲惫、恐惧、绝望统统排出了体外。大脑前所未有的清晰，视力、听力、感知力都在这一刻达到了巅峰。

“爽！”老沙更是直接把自己半个身子都泡在湖里，“电路清洗中……逻辑模块重组中……散热系统效率提升至300%……哦，上帝啊，这感觉就像是刚出厂一样！”

就在三人沉浸在这劫后余生的狂喜中时，一阵奇怪的“嗡嗡”声引起了萧千卷的注意。

他抬起头，看向湖边那几棵巨大的光纤树。
在那些发光的枝桠之间，似乎有什么东西在闪烁。

他站起身，带着几分醉意（数据过载带来的微醺感），摇摇晃晃地走过去。

走近一看，他那双被文气洗涤过的眼睛猛地睁大。
只见在那粗壮的藤蔓上，挂着一串串晶莹剔透、散发着紫红色光晕的果实。它们外形酷似葡萄，但每一颗里面都仿佛封印着一个小小的宇宙，流转着绚丽的星云。

萧千卷伸出手，轻轻摘下一颗。
果实落入掌心，微微发烫。

“这是……”
他凑近细看，在那半透明的果皮之下，竟然隐约可以看到一个个微小的汉字在游动、排列、组合。

“葡萄……美酒……夜光杯……”
萧千卷轻轻念出了那些文字。

下一秒，手中的“葡萄”突然破碎，化作一道紫色的流光，直接钻入了他的眉心。

轰！
一股苍凉、悲壮、豪迈的意境，瞬间在他的脑海中炸开。伴随着急促的琵琶声和战马的嘶鸣声，一副千年前边塞战场的画卷，在他眼前徐徐展开。

萧千卷踉跄后退两步，脸上瞬间涌起一抹异样的潮红，眼神却亮得惊人。
他转过身，看着还在湖边狂饮的墨染和老沙，举起手中抓着的另一串紫色光团，放声大笑：

“哈哈哈！墨染！老沙！别喝那没味道的水了！”
“来尝尝这个！这才叫——真正的盛唐！”

（第二百一十章·完）
