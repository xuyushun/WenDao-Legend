# 第二百五十三章 开明兽：生物识别门禁

当三人真的做到了“心无杂念”，将自身的数据密度降到最低时，奇迹发生了。
那光秃秃的驾驶舱竟然真的飘了起来，悬浮在黑色的弱水之上。

老者点点头，竹竿一点：“走着。”
那根脊骨化作一道流光，牵引着驾驶舱，瞬间划过数十公里的黑暗水域，抵达了彼岸。

彼岸，便是昆仑山门。
这里没有想象中的仙气缭绕，只有九座巨大的、金属铸造的关隘，环绕着登山的主路。
而在山门正中央，蹲着一只庞然大物。

它有着虎的身躯，但脖子上长着九个脑袋。
这九个脑袋不是血肉，而是九个不同制式的摄像头和传感器阵列：红外、声呐、量子雷达、热成像、面部识别、虹膜扫描、脑波读取、因果律测算、灵魂鉴定。

【身份：开明兽】
【职能：昆仑山正门生物识别门禁系统】
【权限：拒绝一切未经授权的访问】

“我就送你们到这儿了。”老者收起竹竿，“这一关，不仅要刷脸，还得刷心。祝好运。”
说完，老者消失在弱水迷雾中。

萧千卷三人刚踏上岸。
开明兽的九个脑袋同时转了过来。十八道红光死死锁定了他们。

“访问请求：拒绝。”
正中间那个最大的脑袋（主控端）发出冰冷的电子音。
“检测到非法入侵者。通缉犯：萧千卷。威胁等级：高。”
“执行：驱逐。”

还没等三人说话，开明兽周围的九座关隘同时开启，无数自动防御机炮伸了出来。

“等等！”萧千卷大喊，“我有通行证！”

他拿出了那枚兵家虎符。
“这是大秦……”

“大秦权限在昆仑无效。”开明兽冷冷地打断，“这里是独立服务器，不归咸阳管。就算是始皇帝来了，没有密钥也进不去。”

“那这个呢？”
萧千卷心一横，掏出了另一件东西。
那是在长平战场，赵括残魂消散前，在那光雨中留下的一块古旧的玉佩。
那是更久远的、属于周朝的信物——穆天子当年留给造父的“御车令”。

开明兽的九个脑袋突然停住了。
中间那个脑袋凑近了那块玉佩，红色的光线反复扫描。

“检测到管理员签名……”
“签名者：姬满（周穆王）。”
“权限级别：至尊VIP。”
“指令：放行……等等，签名已过期两千年。”

开明兽抬起头，九个脑袋露出一种拟人化的戏谑表情。
“虽然是真货，但版本太旧了。想进去？可以。按规矩，回答我三个问题。只要有一个脑袋觉得你在撒谎，就地处决。”

（第二百五十三章·完）
