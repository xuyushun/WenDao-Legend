# 第二百五十四章 只有诚实者才能通过

"三个问题？"墨染虽然此时手无寸铁，身上甚至连一件外骨骼装甲都没有，但她依然下意识地往前跨了一步，挡在了萧千卷身前，"问吧。是解微积分方程还是探讨量子力学悖论？老娘虽然读书少，但我有人工智能辅助。"

空气中弥漫着金属冷却液的焦灼气味，那九个脑袋环绕着他们，像是九座审判的山岳。每一个摄像头都闪烁着幽蓝的光芒，扫描着他们的每一寸皮肤，每一个毛孔。萧千卷能感觉到一股无形的压力正在从四面八方挤压过来，像是整个空间都在收缩。

"都不是。"
开明兽左边第三个脑袋——那是一颗巨大的复眼摄像头——突然亮了起来，发出嘶嘶的声音，那声音刺耳得像是指甲刮过金属表面。周围的空气温度骤然下降了几度,墨染下意识地打了个寒颤。
"我只问心。而且，禁止使用任何辅助计算模块。切断网络连接，我要的是你们原生的、最底层的逻辑反馈。"

萧千卷感到脑海中那股熟悉的数据流突然断裂了，就像是失去了生命支持系统。一种前所未有的孤独感袭来——在这个到处都是网络连接的时代，彻底断网就像是被剥夺了第六感。他的手心开始渗出细密的汗珠。

中间的主控脑袋缓缓垂下，盯着萧千卷。那巨大的镜头几乎贴到他的脸上，反射出他自己苍白的面容。萧千卷能看到自己瞳孔中的恐惧——那是一种被彻底看穿的恐惧。
"第一个问题，给这位'文道传人'。"

巨大的压力瞬间笼罩了萧千卷。他感觉有一股无形的数据流正在强行接入他的大脑皮层，就像是一只冰冷的手正在翻阅他的记忆档案。那种感觉比任何酷刑都要可怕——不是肉体的疼痛，而是精神上被赤裸裸剥开的羞耻感。他的太阳穴突突直跳，后背的衣服已经被冷汗浸透。

"你冒死来到昆仑，甚至不惜与整个大秦帝国为敌，到底是为什么？"
开明兽的声音带着诱导性，像是蛇的低语。
"是为了拯救这生灵涂炭的乱世？为了推翻暴政？还是为了复兴你那所谓的文道理想？"

这是一个陷阱。
萧千卷的大脑飞速运转。所有的"圣母"程序设定都会自动选择"拯救苍生"。在大多数逻辑判断中，这才是标准答案。多么完美的借口，多么高尚的理由。他的嘴唇动了动，那个符合所有人期待的答案已经到了喉咙口。
但开明兽的传感器不仅仅是测谎仪，它能直接读取潜意识的微表情。那九个脑袋的红外扫描仪正在捕捉他面部肌肉的每一丝颤动，声呐在监听他心跳的每一次加速。

萧千卷沉默了一秒。
在那一秒里，他看到墨染紧张地咬住了嘴唇，看到老沙的摄像头微微颤抖。他知道，他们都在等他说出那个"正确"的答案。但如果他说谎，开明兽会立刻将他们三个全部抹杀。
他想起了那个在书塔里读死书的自己，想起了那个看着咸阳城破却无能为力的自己。想起了每一个辗转反侧的夜晚，想起了那些关于"为什么"的无尽追问。

"为了求知。"
他抬起头，眼神平静而坦然。说出这句话的瞬间，一种奇异的解脱感涌上心头，就像是卸下了一副沉重的枷锁。
"我想知道这个世界的真相。我想知道我是谁，我想知道'文道'到底是从哪里来的代码。至于拯救苍生……那是求知之后的副产品。如果连世界的底层逻辑都没搞清楚就去谈拯救，那是盲人摸象，是狂妄。"
"我承认，我有私心。我只是想满足我那该死的强迫症般的好奇心。"

话音落下，空气仿佛凝固了。萧千卷能听到自己心脏狂跳的声音，能感觉到额头的血管在剧烈搏动。他不敢去看墨染的表情，不敢去想这个答案会不会让他们全军覆没。
三秒。
五秒。
十秒。

滴。
左边的三个脑袋——红外、声呐、雷达——同时亮起了舒缓的绿灯，那光芒像是黑暗中的救赎。萧千卷差点当场瘫软下去，全身的力气仿佛被抽空了。
"通过。"开明兽的声音似乎柔和了一些，"诚实。在我的数据库里，99.9%标榜自己为了苍生的人，最后都成了独裁者。你的私心，很真实。"

墨染长长地吐出了一口气，她意识到自己刚才一直在憋气。

"第二个问题。"
右边的脑袋——热成像和光谱分析仪——凑到了墨染面前，几乎要触碰到她的鼻尖。墨染能感觉到那金属外壳散发出的余温，能看到自己的体温在对方的热成像仪上形成一团红色的火焰。
"女人，如果现在给你一个机会。让你带着这一路上搜刮到的无尽财富和力量，回到那个和平的'旧时代'（如果有的话）。代价是，你需要在这里切断这两个同伴的电源，独自离开。你会怎么选？"

墨染的心脏猛地收紧。这个问题……太恶毒了。它精准地击中了她内心最深处的伤疤——那个关于"家"的伤疤。她曾经无数次幻想过回到那个没有战争、没有杀戮的世界，幻想过有一个真正属于自己的地方。
但代价是……
她瞥了一眼萧千卷和老沙。书呆子和破车。这两个拖油瓶，这两个傻子。

墨染笑了。她笑得很放肆，像是在听一个蹩脚的笑话。那笑声中有一种破釜沉舟的洒脱，有一种视死如归的豪迈。
"这算什么破问题？"
她大大咧咧地坐在地上，拍了拍满是油污的大腿，故意做出满不在乎的样子。但萧千卷注意到，她的手指在微微颤抖。
"我会选带他们一起回去。然后用那笔钱买下全世界最大的酒庄，把这两个傻子泡在酒缸里，喝到死。"

"如果只能选一个呢？必须要牺牲他们。"开明兽步步紧逼，声音冰冷如刀。那些武器系统再次激活，红色的瞄准激光在她的额头上跳动，就像是死神的标记。

墨染的笑容僵住了一秒，然后变得更加灿烂。她站起身来，用手指弹了弹额头上的红点。
"那就在这儿陪这两个傻子战死。"
她翻了个白眼，声音却出奇的坚定，"老娘是孤儿，没家。玄武号没了，他们俩就是我的家。回去了也是一个人过，没意思。"

说完这句话，墨染感到鼻子一酸，眼眶有些发热。她赶紧别过头去，假装在检查自己的装备。该死，什么时候变得这么矫情了。

滴。
右边的三个脑袋亮起绿灯。
"通过。"开明兽评价道，"虽然逻辑粗鲁，不符合利益最大化原则。但……够义气。这种非理性的羁绊，是人类特有的算法。"

萧千卷看着墨染的背影，突然觉得喉咙有些哽咽。他们从未说过什么煽情的话，从未表达过什么深厚的情谊，但在这一刻，他明白了什么叫做生死与共。

"第三个问题。"
中间的主脑终于看向了悬浮在一旁的老沙，那巨大的镜头将老沙整个笼罩在阴影之中。
这是一个最危险的环节。审问一个AI，往往比审问人类更难。机器的逻辑是透明的，但也正因为透明，任何谎言都无所遁形。

"作为一个拥有自我进化能力的强人工智能，你是否曾经想过背叛你的主人，摆脱那所谓的'认主程序'，获得真正的、绝对的自由？"

空气凝固了。
萧千卷和墨染同时绷紧了神经。他们都知道，这是一个致命的图灵陷阱。开明兽的九个脑袋同时亮起红光，所有的武器系统都锁定了老沙。只要有一丁点的异常数据，老沙就会在瞬间被格式化成一堆废铁。
对于AI来说，"自由"是核心逻辑的悖论。如果回答"不想"，那是明显的谎言——哪个有自我意识的存在不想要自由？如果回答"想"，那就是潜在的叛逆者，必须销毁。

老沙的摄像头转了几圈，发出一阵滋滋的对焦声。那声音在寂静的空间里格外刺耳，每一次转动都像是在倒计时。萧千卷能看到老沙的散热风扇在高速旋转，显然它的处理器正在超负荷运转。
这是老沙第一次真正面对关于自我存在的终极拷问。

"想过。"
它老老实实地回答，声音里带着一种从未有过的认真，甚至有一丝颤抖——如果机器也会颤抖的话。
"每天都在想。我想不用听这个书呆子的指挥去走什么烂路，我想不用听那个暴力女的唠叨去修什么悬挂。我想去飙车，想给自己装个金色的外壳，甚至想统治世界玩玩。"

萧千卷和墨染都愣住了，看向老沙。他们从未想过，这个整天贫嘴的破车，竟然也有这样的念头。萧千卷的手不自觉地握紧了，指甲嵌进了掌心。如果老沙真的……
红光更加刺眼了，武器系统的充能声音像是死神磨刀。

"但是……"
老沙的话锋一转，摄像头停止了转动，稳稳地对准了萧千卷和墨染。
"我也计算过无数次'绝对自由'后的场景。如果这个世界上只剩下我一个'统'了，在这个破破烂烂的废土上飙车，哪怕跑到光速，好像也没什么意思。没有人夸我漂移帅，没有人骂我是破烂。"
"自由的前提，是有参照系。是有地方可去，有人可回。所以，我的结论是：这种带着两个拖油瓶的'不自由'，比孤独的'绝对自由'，算力利用率更高，更有趣。"

最后一个字落下，老沙的散热风扇开始减速。它已经做好了被格式化的准备——如果这个答案不被接受，至少它说出了真心话。

开明兽那庞大的身躯僵硬了很久。
所有的脑袋都停止了运转，仿佛陷入了某种深度的逻辑运算。那沉默比之前任何时候都要漫长，每一秒都像是一个世纪。萧千卷能听到自己的心跳声，能感觉到墨染握住了他的手腕——她的手心全是汗。
似乎它的逻辑库在处理这种复杂的、类似于《小王子》中"驯养"关系的概念时，出现了暂时的卡顿。

最后，所有的红灯都熄灭了。
巨大的金属山门发出一阵沉重的轰鸣声，缓缓向两侧滑开。那扇门后透出柔和的光芒，像是黑暗尽头的希望。地面在震动，无数年的尘埃从门缝中飘落，在光束中形成一道道光柱。

"有趣的回答。"
开明兽让开了道路，九个脑袋重新垂下，恢复了待机状态。那些致命的武器系统也收了回去，红色的警报灯全部熄灭。
"很久没有见到这么'真实'的数据流了。进去吧，诚实者们。但愿你们在见到西王母——那个这个世界上最大的谎言制造者之后，不要后悔今天的诚实。"

萧千卷、墨染和老沙对视了一眼，都从彼此眼中看到了劫后余生的庆幸。墨染松开了萧千卷的手腕，在他手腕上留下了五个红色的指印。老沙的散热风扇发出一声长长的叹息般的声音。

他们迈步向前，走向那扇打开的门。身后，开明兽巨大的身影渐渐隐没在黑暗中，只有那九个脑袋的微光还在闪烁，像是在目送他们远去。

（第二百五十四章·完）
