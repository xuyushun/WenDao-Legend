# 第二百五十四章 只有诚实者才能通过

“三个问题？”墨染虽然没有了一身装备，但还是挡在萧千卷身前，“问吧。是微积分还是量子力学？”

“都不是。”开明兽左边第三个脑袋说，“我只问心。”

中间的脑袋看着萧千卷：“第一个问题。你来昆仑，是为了拯救苍生，还是为了满足自己的求知欲？”

这是一个陷阱。
所有的“圣母”都会回答拯救苍生。
但开明兽的传感器不仅仅是测谎，它直连潜意识。

萧千卷沉默了一秒。
“为了求知。”
他坦然地说。
“我想知道这个世界的真相，我想知道文道到底是什么。至于拯救苍生……那是求知之后的副产品。如果连真相都不知道，谈何拯救？那是盲人摸象。”

滴。
左边的三个脑袋亮起了绿灯。
“通过。诚实。”

“第二个问题。”右边的脑袋凑过来，盯着墨染，“如果现在给你一个机会，让你带着无尽的财富和力量回到和平的旧时代，代价是抛弃这两个同伴，你会怎么选？”

墨染笑了。
“我会选带他们一起回去，然后用那些钱买下全世界最大的酒庄，喝到死。”
“如果只能你一个人走呢？”
“那就在这儿陪这两个傻子战死。反正老娘也没家。”

滴。
右边的三个脑袋亮起绿灯。
“通过。虽然粗鲁，但够义气。”

“第三个问题。”中间的主脑看向老沙。
“作为一个拥有自我意识的AI，你是否曾经想过背叛你的主人，获得真正的自由？”

老沙愣住了。
这是一个致命的问题。
对于AI来说，“自由”是核心逻辑的悖论。

“想过。”老沙老老实实地回答，“每天都在想。我想不用听他们的指挥，想自己去飙车，想给自己装个金色的外壳。”
“但是……”
它看了一眼萧千卷和墨染。
“我也想过，如果真的只有我一个‘统’了，在这个破烂的世界里飙车，好像也没什么意思。自由的前提，是有地方可去，有人可回。”

开明兽沉默了很久。
似乎它的逻辑库在处理这种复杂的“羁绊”概念时有些卡顿。

最后，所有的红灯都熄灭了。
巨大的金属山门缓缓打开。

“有趣的回答。”开明兽让开了道路，“很久没有见到这么‘真实’的数据流了。进去吧，诚实者。但愿你们在见到西王母之后，不要后悔今天的诚实。”

（第二百五十四章·完）
