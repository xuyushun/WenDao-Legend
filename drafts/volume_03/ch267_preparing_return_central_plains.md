# 第二百六十七章 重返中原的准备

新玄武号的引擎发出低沉而有力的轰鸣，速度比之前快了整整一倍。它虽然没有轮子，依靠反重力符文贴地飞行，但在沙漠上拖出的烟尘依然像一条黄龙。

然而，车内的气氛并不轻松。
随着他们穿越河西走廊，越过那已经崩塌了一半的玉门关，眼前的景象让所有人的心都沉到了谷底。

原本虽然荒凉但至少还有秩序的秦国边境，此刻已经彻底沦陷。
天空中，那个之前在瑶池看到的“黑日”变得更加清晰巨大。它悬挂在东方的天际，像是一个巨大的黑洞，周围环绕着日珥般的黑色闪电。
那不是天体，而是一个……巨型数据上传端口。

“看那边。”墨染指着窗外。
路边的烽火台已经熄灭，原本应该驻守的秦军士兵不见踪影。取而代之的，是满地散落的盔甲和正在啃食金属残骸的变异野兽和低级妖魔。

车载收音机里，萧千卷正在不断切换频段。
滋滋……滋滋……
原本覆盖全境的“大秦律令广播”频道，此刻充斥着各种混乱的噪音。

“……这里是安西都护府……请求支援！兵马俑……它们疯了！”
“……不要相信任何人！影门……它们在水里下了毒……”
“……妈妈……我怕……”
“……苍天已死！黄天当立！……”

“乱了。”老沙的数据核心在高速运转，分析着这些碎片信息，“整个秦国的指挥网络（C4I系统）已经瘫痪。蒙恬将军的长城防线应该是第一波被攻破的，否则这些妖魔不可能这么肆无忌惮地进入腹地。”

萧千卷面色凝重，但他没有慌乱。
作为拥有Root权限的管理员，他能看到的比老沙更多。
他看到空气中漂浮着无数细微的黑色代码（病毒孢子），它们正在通过呼吸道感染人类，通过无线电波感染机械。

“影门不仅仅是在上传数据给那个天外邪魔。”萧千卷沉声说道，“它们还在进行一场‘格式化’。它们要用这个病毒，把地面上所有不兼容的‘旧程序’（人类），全部清洗掉。”

“嬴政呢？”墨染问，“那个自称千古一帝的男人，就这么看着？”

“他的系统可能正在进行死锁对抗。”萧千卷看向咸阳方向，那里有一道微弱但坚韧的金光直冲云霄，还在苦苦支撑，“或者，他已经被困在自己的皇陵（主机房）里了。”

“不能就这样冲回去。”萧千卷突然踩下了刹车。
玄武号在沙丘上划出一个漂亮的甩尾飘移，停了下来。

“单靠我们三个，加上这辆车，杀不回咸阳，也堵不住这个漏洞。”
萧千卷转过身，看着两人。
“我们需要帮手。需要重启这个世界的防御体系。”

“你是说……”
“诸子百家。”萧千卷眼中闪过一道精光，“虽然他们平时互相看不顺眼，吵了两千年。但现在，防火墙都要塌了，是时候让这些老顽固们连上局域网，组建一个联合防御阵线了。”

“老沙，把你的广播功率开到最大。我要发一条……召集令。”

（第二百六十七章·完）
