# 第二百六十七章 重返中原的准备

重返中原的路上，气氛变得格外压抑。
因为随着他们越过西域的边界，进入秦国的疆域，发现情况比他们想象的还要糟糕。

原本虽然严酷但井然有序的秦国网络，此时已经是一片混乱。
长城防线依然在闪烁，但那是报错的红光。
不仅是西边的缺口没堵上，其他地方也出现了大面积的崩塌。
更可怕的是，天空中不知何时多了一轮“黑色的太阳”。

“那是病毒聚合体。”萧千卷看着那个黑日，“影门启动了某种献祭仪式，正在把地面的数据强行上传给那个天外邪魔。”

“秦国怎么没动静？”墨染不解，“蒙恬呢？嬴政呢？”

“他们可能……自身难保。”
萧千卷打开车载收音机。
原本全是官方通报的频道，现在只剩下刺耳的杂音，偶尔夹杂着几声求救信号。

“这里是咸阳……请求支援……兵马俑失控了……”
“这里是琅琊台……徐福疯了……”
“这里是……滋滋……”

“大秦乱了。”萧千卷关掉收音机，“这是一个机会，也是一个灾难。乱世出英雄，但也出疯子。”

他看向老沙。
“老沙，帮我发个信号。不是发给秦国，是发给全天下的‘读书人’。”
“你要干什么？”
“召集盟友。单靠我们要对抗那个黑日是不够的。我们需要重启‘诸子百家’的局域网，组建一个联合防火墙。”

（第二百六十七章·完）
