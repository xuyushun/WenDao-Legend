# 第三百二十三章 神农代码

紧接着到达的，是农家。

那几辆改装过的核动力拖拉机开进长安城时，发出的轰鸣声简直比坦克还要震撼。

它们体型巨大，钢铁车身上焊着一层又一层的改装装甲，每一道焊缝都带着某种豪放的粗糙，像是用人力硬生生地把几个不搭调的东西捏成了一件有用的事物。废气管里喷出的热流把冬日的冷空气搅出一圈一圈的白雾，和周围废墟的灰色形成了奇异的对比。后车斗里装的不是炮弹，而是满满当当的黑色土壤样本和各种奇怪的菌种罐子，还有一箱箱用特制防护罩密封的金属种子盒，以及成卷成卷的生长灯管，和一台台不知道是什么原理的农业改造机器。那些拖拉机一字排开地停在城门内侧，发动机缓缓熄火，残存的热浪从机体上散出来，烘得旁边的空气微微扭曲。许行的那帮弟子们从车上跳下来，清一色的粗布衣，沾满了泥土，手掌宽厚如铲子，鞋底厚得像两块砖头——这些人身上每一个细节都在说明，他们是真正在土地上活了一辈子的人。

那个秦岭的壮汉叫许行，人如其名，是个实干派。

他跳下车，靴子踩在地上，发出沉重的一声，震得脚下细碎的瓦砾轻轻跳了一跳。他甚至没去和墨染打招呼，直接蹲下身子，把手掌按在长安城外那片污染土地上。片刻后，他捡起一把泛着金属光泽的黑泥，放在鼻尖闻了闻，又把一小撮放在舌尖轻轻地抿了抿，眉头拧成了一个疙瘩。旁边的墨染看得目瞪口呆——她没见过有人用这种方式检测土壤成分，但许行的表情告诉她，这一舌头的情报，比任何精密仪器给出的数据都要直接。

那土的气味是刺鼻的——金属气息过重，像是把一把铁锉磨成的粉末揉进了土里，底下还有一股像是溶剂一样的化学气味，刺得鼻腔发酸。这土是被系统的工业污染和辐射残留双重破坏的结果，哪怕是再顽强的野草，在这里也只能生长出扭曲的、奇形怪状的变异株，根本不能用于种植。

"这土不行。"

他把土扔掉，拍了拍手，"重金属超标五百倍，酸碱度极度失衡。在这种土里种出来的麦子，吃了不用第二天，当晚就会变异长出三个头。"

"那怎么办？"公输盘负责接待，看着那几万张等着吃饭的嘴，急得团团转，"我们现在的食物储备只够撑到明天早上了。要不……种点耐辐射的苔藓？"

"苔藓？那玩意儿吃了拉嗓子。"

许行一脸嫌弃地瞥了他一眼，"既然土不行，那就改土。"

他一挥手，那十几台拖拉机轰隆隆地开进了荒地。

它们尾部的喷洒装置依次打开，向地下注入了一种粘稠的、散发着淡淡荧光的绿色凝胶。那凝胶落地时无声无息，但触地之后，却开始以一种微妙的方式向泥土深处渗透，像是一场极其耐心的渗透，一点一点地渗进那些板结的土层缝隙里。

"这是'息壤'二号。"许行解释道，声音里带着那种庄稼汉对自家庄稼的深沉自豪，"是我们农家在秦岭地下培育了整整五十年的转基因菌群。它们不吃别的，专门吃重金属、吃塑料微粒、吃辐射残留。吃完了就拉出有机肥。"

哪怕是公输盘这种见惯了黑科技的人，也被眼前的景象惊呆了。

没有一声巨响，没有任何炫目的光效，甚至近乎悄无声息。但在那片被喷洒过凝胶的荒地上，变化在发生，能感觉得到，却说不清是从哪一刻开始的。就像是春天的第一声蛙鸣，等你意识到的时候，已经是一片蛙声大合唱了。

最先察觉到的是嗅觉。那股刺鼻的金属气息和化学溶剂味，像是被一只无形的手缓缓抹去，从浓烈到淡薄，最终几乎消散殆尽。取而代之的，是那种三百年没有人在这片土地上嗅闻过的气息——泥土的气息，湿润的，带着一点腐殖质的甜味，像是被雨水淋透的田野，像是祖辈留下来的记忆里才有的那种安心的味道。

短短几个小时，原本死气沉沉、板结干裂的土地，竟然开始像呼吸一样起伏。那是土地被激活时的细微膨胀与收缩，像是某种沉睡了三百年的东西正在苏醒。土地变得松软、湿润，甚至能看到有几条机械改造过的蚯蚓从深处钻出来，在地面上打了个滚，然后又钻回去。

围在外围的幸存者们没有人说话。有人不自觉地向前走了两步，停在田埂边，弯腰去摸了摸那片泥土。手指陷入土里的一瞬间，传来了松软、湿热的触感，那是活土才有的感觉。有个上了年纪的老人，眼眶突然就红了，他把手心贴在土上，像是在摸什么珍贵的东西，嘴唇在颤抖，却没有发出声音。

许行在田埂边蹲了整整三个小时，寸步未离，把那片正在慢慢苏醒的土地当成了自己的孩子——就像他这辈子所有的孩子，都是这片土地。他的手掌不时地按在泥土上，感受着那些细微的温度和湿度变化，眼神里有一种外人很难理解的专注，那是农人与土地之间最古老的对话，不需要数据，不需要仪器，只需要一双经验丰富的手，和一颗与这片土地血脉相连的心。

"播种！"许行大吼一声。

一排排无人机掠过低空，嗡嗡的旋翼声压低了所有人的声音，撒下了那些幽幽蓝光的种子。

这一次，展现的是真正的奇迹。

不需要几个月，甚至不需要几周。在一种只有农家才知道频率的生长灯的高频照射下，种子破土而出。抽芽、长叶、拔节、开花、灌浆——这一切仿佛是被按下了快进键，仿佛时间在这片土地上有了不同的流速。

几个小时后，一片金黄色的麦浪，在曾经是废墟的土地上翻滚。

那一刻，没有人说话。风从远处低低地吹过来，掀起麦穗轻柔的波浪，麦芒在光线下闪烁着细碎的金色，麦香四溢，弥漫开来——那是谷物在阳光和泥土的催化下生长出来的天然气息，是这个世界上最古老、最朴素的幸福的气味，是那种三百年来只存在于被系统删除的古老记录里的味道。周围的幸存者们呆站在田埂边，谁也没说话。有人慢慢蹲下来，把鼻子凑近一株刚刚灌浆的麦穗，深深地吸了一口气，然后就再也没有站起来，只是蹲在那里，久久地不动。有几个年轻人是系统控制下长大的，从来不知道麦子长什么样——他们最初闻到麦香，居然本能地以为是什么设备泄漏了什么气体，下意识地后退了几步，直到旁边的老人拉住了他们的手，把那种气息的名字告诉他们：这叫麦香，这叫庄稼地，这叫……活着。

"这也太违反自然规律了吧？"正在旁边围观的难民们眼珠子都快掉下来了。有个系统管理员出身的人，摸出了一块便携式检测仪，对着那片麦田扫描了半天，最后把仪器塞回去，沉默了良久，"……检测结果显示是真实有机物。不是全息投影。这……这是怎么做到的？"

"这就是自然规律。"许行看着这片麦田，憨厚的脸上露出了如同父亲看孩子般的笑容，"万物生长靠太阳。但如果太阳不够，我们就自己造个太阳；如果时间不够，我们就把时间压缩进代码里。这就是我们农家的'神农代码'。只要有这代码在，华夏人就饿不死，民心就不会散。"

当天晚上，长安城的广场上架起了几百口大锅。

烟火气腾腾地升起，飘散在这座沉默了三百年的废墟城市上空。

这不是合成蛋白块，也不是过期罐头。是真正的、刚刚收割磨成粉、热气腾腾的白面馒头。那种谷物特有的香甜气息，混合着柴火燃烧的烟气，从大锅边一圈一圈地扩散出去，弥漫在整个广场，穿过每一条巷子，钻进每一处角落。很多人拿着馒头的手都在颤抖，舍不得下嘴。有老人把馒头托在掌心，就那么看着，泪水不知什么时候流了下来，嘴里喃喃着什么，却听不清楚，也许是在叫一个很久很久没有叫过的名字。有个孩子把馒头在鼻子底下嗅了又嗅，然后认真地宣布："这个是真的好闻。"旁边的大人轻轻地把他揽进怀里，什么也没说，只是抱着。

墨染刚刚从端木蓉的脑部手术中醒过来，头上还缠着纱布，脑袋里还有些发懵。她被人塞了一个馒头。

她一口咬下去，眼泪差点掉下来。

虽然因为生长周期太短，口感稍微有点像嚼蜡，没有那种自然的劲道。但对于这些吃了半辈子化学合成膏、甚至连这就是"食物"都不知道的人来说，这就是世界上最美味的佳肴，是活着的味道，是一种被这片土地接纳和喂养的踏实感。

"真香啊……"墨染一边吃一边含糊不清地说。

她想起了许多事——那些在地下城吃着压缩饼干对她说"等自由了吃顿好的"、最后却没等到那一天的人；想起了萧千卷曾经说过，等长安光复了要去吃一顿地道的陕西面，加很多辣子那种。那些话现在想来，轻描淡写，却有千斤重。

这一刻，长安城才真正有了"人气"。

有了医，能救命；有了食，能续命。就有了哪怕天塌下来，这群人也能笑着活下去的底气。那种底气不是来自武器，不是来自数量，而是来自这个很简单、很古老的事实：有人治病，有粮可吃。人活着，就有希望。

许行抱着双臂，靠在拖拉机边上，看着广场上升起的炊烟，脸上是那种喝了一口水才发现自己已经渴了很久的满足感。他把一只粗糙的大手搭在那台四十年的拖拉机上，轻轻拍了拍，就像拍一匹陪他走了很远的老马。

"行了，咱们没白跑这一趟。"

（第三百二十三章·完）
