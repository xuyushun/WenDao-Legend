# 第三百二十三章 神农代码

紧接着到达的，是农家。
那个秦岭的壮汉叫许行（农家代表人物名）。他带来的不是枪炮，而是拖拉机和种子。

“这土不行。”
许行抓起长安废墟的一把土，闻了闻。
“重金属超标五百倍，种出来的麦子吃了会变异长出三个头。”

“那怎么办？我们几万人等着吃饭呢。”公输盘问。
“改土。”
许行一挥手。
他带来的那十几台拖拉机轰隆隆地开进了荒地。
但它们并没有犁地，而是向地下注入了一种绿色的凝胶。
“这是‘息壤’二号。一种能够吞噬重金属并将其实转化为有机肥的转基因细菌群。”

短短几个小时，原本黑乎乎、散发着恶臭的土地，竟然变成了肥沃的黑土。
然后播种。
那些发着蓝光的种子一下地，就开始疯狂生长。
不需要几个月，只需要几天。
抽芽、长叶、开花、结穗。
看着那金黄色的麦浪在废墟上翻滚，所有人都看呆了。

“这也太违反自然规律了吧？”
“这就是自然规律。”许行憨厚地笑了，“万物生长靠太阳。我们只是把太阳的能量通过更高效的代码写入了基因里。这就是‘神农代码’。”

当天晚上，长安城吃上了第一顿真正的、热气腾腾的白面馒头。
虽然味道有点怪（因为生长太快，口感稍微有点像塑料），但对于这些吃了半辈子合成膏的人来说，这就是世界上最美味的佳肴。

墨染（刚醒过来就被塞了一个馒头）一边吃一边流泪。
“真香。”

这一刻，长安城才真正活了过来。
有了医，有了食。
就有了哪怕天塌下来也能活下去的底气。

（第三百二十三章·完）
