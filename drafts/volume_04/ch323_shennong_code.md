# 第三百二十三章 神农代码

紧接着到达的，是农家。
那几辆改装过的核动力拖拉机开进长安城时，发出的轰鸣声简直比坦克还要震撼。车斗里装的不是炮弹，而是满满当当的黑色土壤样本和各种奇怪的菌种罐子。

那个秦岭的壮汉叫许行，人如其名，是个实干派。
他跳下车，甚至没去和墨染打招呼，直接蹲在长安城外那片被辐射严重污染的荒地上。
他抓起一把泛着金属光泽的黑泥，放在鼻尖闻了闻，眉头拧成了一个疙瘩。
“这土不行。”
他把土扔掉，拍了拍手，“重金属超标五百倍，酸碱度极度失衡。在这种土里种出来的麦子，吃了不用第二天，当晚就会变异长出三个头。”

“那怎么办？”公输盘负责接待，看着那几万张等着吃饭的嘴，急得团团转，“我们现在的食物储备只够撑到明天早上了。要不……种点耐辐射的苔藓？”

“苔藓？那玩意儿吃了拉嗓子。”
许行一脸嫌弃地瞥了他一眼，“既然土不行，那就改土。”

他一挥手，那十几台拖拉机轰隆隆地开进了荒地。
它们尾部的喷洒装置打开，向地下注入了一种粘稠的、散发着淡淡荧光的绿色凝胶。
“这是‘息壤’二号。”许行解释道，“是我们农家在秦岭地下培育了整整五十年的‘转基因甚至菌群’。它们不吃别的，专门吃重金属、吃塑料微粒、吃辐射残留。吃完了就拉出有机肥。”

哪怕是公输盘这种见惯了黑科技的人，也被眼前的景象惊呆了。
短短几个小时，原本死气沉沉、板结干裂的土地，竟然开始像呼吸一样起伏。黑色的恶臭逐渐消散，取而代之的是一种久违的泥土芬芳。
土地变得松软、湿润，甚至能看到有蚯蚓（也是机械改造过的）在翻身。

“播种！”许行大吼一声。
一排排无人机掠过低空，撒下了那幽幽蓝光的种子。
这一次，展现的是真正的奇迹。
不需要几个月，甚至不需要几周。
在一种只有农家才知道频率的生长灯照射下，种子破土而出。
抽芽、长叶、拔节、开花、灌浆。
这一切仿佛是被按下了快进键。
几个小时后，一片金黄色的麦浪，在曾经是废墟的土地上翻滚。风吹过，麦香四溢。

“这也太违反自然规律了吧？”正在旁边围观的难民们眼珠子都快掉下来了。
“这就是自然规律。”许行看着这片麦田，憨厚的脸上露出了如同父亲看孩子般的笑容，“万物生长靠太阳。但如果太阳不够，我们就自己造个太阳；如果时间不够，我们就把时间压缩进代码里。这就是我们农家的‘神农代码’。只要有这代码在，华夏人就饿不死。”

当天晚上，长安城的广场上架起了几百口大锅。
这不是合成蛋白块，也不是过期罐头。
是真正的、刚刚收割磨成粉、热气腾腾的白面馒头。
那种谷物特有的香甜气息，让很多人拿着馒头的手都在颤抖，舍不得下嘴。

墨染刚刚从端木蓉的脑部手术中醒过来，头上还缠着纱布。他被塞了一个馒头。
他一口咬下去，眼泪差点掉下来。
虽然因为生长周期太短，口感稍微有点像嚼蜡，没有那种自然的劲道。
但对于这些吃了半辈子化学合成膏、甚至连这就是“食物”都不知道的人来说，这就是世界上最美味的佳肴，是活着的味道。

“真香啊……”墨染一边吃一边含糊不清地说。

这一刻，长安城才真正有了“人气”。
有了医，能救命；有了食，能续命。
就有了哪怕天塌下来，这群人也能笑着活下去的底气。

（第三百二十三章·完）
