# 第三百七十六章 格式化 C:

赵高的“尸体”还在地上散发着焦臭味。
但仅仅过了三秒，整个咸阳城——不，是整个连接着“秦网”的世界——突然陷入了一片死寂。

灯光熄灭了。
风声停止了。
就连空气中原本无处不在的数据流动声（那是赛博时代的背景白噪音）也消失了。

“怎么回事？”
李太白警惕地环顾四周，“那老太监还没死透？”

“不，他死透了。”萧千卷看着手中的天问剑，剑身上的光芒因为失去连接而开始忽明忽暗，“麻烦就在于，他死得太透了。”

“警告！警告！系统管理员信号丢失！”
“警告！维护进程中断！”
“触发最高级别防御协议：【焚书坑儒 2.0】。”
“逻辑判定：既然没有合法的管理员，那么所有现存数据皆视为……病毒。”

老沙的终端突然炸出一团火花，他惊恐地尖叫起来：“疯了！系统疯了！它正在执行全盘格式化！它要删除一切！”

天空中，原本紫色和绿色的霓虹天幕，变成了一片惨白。
不是白云，是那种显示器坏掉时的死白。
这片白色像是一场雪崩，从天顶开始向下蔓延。
凡是被白色覆盖的地方，无论是建筑、树木，还是没来得及逃跑的机械士兵，都在瞬间分解成了最原始的二进制代码——0和1，然后消散在虚空中。

“这是……数据坍缩。”韩非的脸色从未如此凝重，“系统判定世界不再安全，所以它决定‘重启’。就像电脑中毒后不仅杀毒，还要把硬盘格式化一样。”

“格式化？”墨染看着远处已经消失了一半的渭水，“你是说，全人类都会变成……空白文件？”

“不，是连文件都不剩。”
韩非指着那迅速逼近的白色风暴，“记忆、历史、情感、肉体……所有能被定义为‘信息’的东西，都会被归零。”

“这他娘的也太狠了！”公输盘一拳砸在地上，“赵高那王八蛋，临死还给我们留了个同归于尽的炸弹！”

“这就是‘暴政’的本质。”萧千卷看着那逼近的毁灭，“当权力高度集中于一点，一旦那个点崩溃，整个体系就会为了‘自保’而拉着众生陪葬。”

白色的风暴已经吞噬了咸阳城的外郭。
无数还在庆祝胜利的反抗军战士，还没来得及发出惨叫，就被分解成了无数光点。
那是真正意义上的抹杀。不留痕迹，不留尸骨。

“必须有人挡住它。”
萧千卷握紧了天问剑，准备冲上去。
“师兄，你不行。”
一只手拦住了他。
是韩非。

韩非摘下了鼻梁上的金丝眼镜，仔细地擦了擦，然后重新戴好。
“天问剑只能斩有形之敌，斩不了这种无形的规则。”
“要对抗规则，只能用规则。”

“你要干什么？！”墨染有一种不祥的预感。

韩非没有回答，只是整理了一下衣冠，朝着那漫天的白色风暴，盘膝坐下。
“法家弟子韩非，今日……以身殉道。”

（第三百七十六章·完）
