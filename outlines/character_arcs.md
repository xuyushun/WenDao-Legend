# 角色成长弧光 (Character Arcs)

## 主要角色

### 萧千卷 (Xiao Qianjuan)
*   **核心身份**：穿越者，书院弃徒 -> 儒家圣人候选
*   **修炼体系**：儒道（文气）
*   **第一卷（潜龙在渊）**：
    *   **起点**：九品·训诂境（Ch2觉醒）。
    *   **状态**：初来乍到，利用现代知识魔改儒术。
    *   **境界进度**：九品训诂 → 八品章句（Ch25）→ 七品章句巅峰（Ch50）→ 六品经义初阶（Ch65）。
    *   **关键转折**：【黑水河之战】觉醒"文以载道"的实战能力。
*   **第二卷（江湖远）**：
    *   **目标**：寻找九鼎，探索世界真相。
    *   **成长**：从单纯的"借用"古人诗词，开始尝试"创造"自己的理论（结合科学与玄学）。
    *   **境界突破**：由六品·经义境中期迈向四品·经义境巅峰（触碰著书境门槛）。
    *   **关键事件**：
        *   墨家城：掌握编程/逻辑思维在阵法中的应用。
        *   鬼谷：领悟博弈论与大局观，心性蜕变。
        *   名家城：利用逻辑悖论攻击现实，初步触碰规则之力。

### 墨染 (Mo Ran)
*   **核心身份**：墨家少主，巨子继承人
*   **修炼体系**：墨家机关术（机械改装）
*   **外貌特征**：左臂为机关义肢（刻满铭文），左眼嵌有精密机关义眼（铜齿轮结构，可分析数据）。义眼平时以碎发遮掩。
*   **第一卷**：
    *   **状态**：被家族放逐，寻找自我。
    *   **设定伏笔**：Ch9 展示机关义肢时同步暴露义眼存在（已写入正文）。
    *   **关键转折**：制造出"朱雀"原型机。
*   **第二卷**：
    *   **目标**：清理门户，统合墨家。
    *   **成长**：进一步强化义肢与义眼（Ch9已有基础），从排斥机械飞升到理解"人机合一"。
    *   **装备升级**：获得【墨核】，朱雀机甲升级为2.0版本。
    *   **心理**：更加果断，甚至冷酷，但在萧千卷面前保留柔软。

### 许归田 (Xu Guitian)
*   **核心身份**：农家/医家传人
*   **修炼体系**：农家（植物操控）+ 医家（炼药/生化）
*   **第一卷**：
    *   **状态**：淳朴少年，种田爱好者。
    *   **伏笔**：Ch12 集市场景客串（卖文气灌溉的巨型灵萝卜的农家少年），为卷二正式入队铺垫。
*   **第二卷**：
    *   **目标**：寻找治愈世间疾苦的方法（神农鼎）。
    *   **成长**：直面医家黑暗面（人体实验），信念受到冲击后更加坚定“仁心”。
    *   **能力觉醒**：【万物生长】——不仅能催熟植物，还能控制微生物/细胞。
    *   **关键事件**：云梦泽大爆发，一人奶全团。

### 阿巴 (A-Ba)
*   **核心身份**：书灵（上古文明AI碎片？）
*   **形态**：漂浮的书/弹幕发射器
*   **成长**：
    *   **第一卷**：Ch2 本命书宫出现微妙异动（伏笔），Ch10-11 正式觉醒，只能叫"阿巴"和发表情包。
    *   **第二卷**：吞噬大量古籍后，语言能力恢复，开始吐槽并提供关键资料检索服务。
    *   **秘密**：随着九鼎收集，阿巴的记忆逐渐复苏，似乎与【中央电脑】有关。

### 公孙游 (Gongsun You)
*   **核心身份**：纵横家传人 / 嘴炮王者 / 智囊担当
*   **修炼体系**：纵横家《鬼谷子》
*   **特征**：风流倜傥，毒舌，扇子上写着"以理服人"。
*   **第一卷**：
    *   **伏笔**：Ch12 集市场景客串（摇折扇的年轻公子，扇面写"以理服人"），为卷二正式入队铺垫。
*   **第二卷**：
    *   **目标**：揭开鬼谷真相。
    *   **核心能力**：【捭阖之术】（语言诱导幻觉/自相残杀）。

## 重要配角

### 老疯子
*   **核心身份**：前朝守墓人 / 主角隐形导师
*   **特征**：落魄山脚乞讨的疯老头，念叨上古神语（甲骨文）。
*   **第一卷**：Ch15 出手救主角团脱困，碗碎法家结界。离别时喃喃甲骨文古韵（已植入正文伏笔）。
*   **后续**：身份线索逐步揭露，与世界底层"绝地天通"之秘相关。

### 法家·韩正方（大司寇）
*   **身份**：法家高层，统管刑狱。
*   **第一卷**：Ch15 登场，下达追捕墨染的命令。注意：与县令"李文渊"无任何关系。

### 路人甲/反派
*   **影门刺客**：从单纯的杀手组织，逐渐揭露为“天外邪魔”的爪牙。
*   **鬼谷子（当代码）**：亦正亦邪，可能是个高级AI程序，在观察人类是否值得被拯救。
