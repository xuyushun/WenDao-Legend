# 第一卷大结局：长安保卫战（赛博修仙版）大纲

**核心设定升级**：将传统的文道/百家设定进行“工业化/科幻化”解释，引入“文明等级”概念。
> **⚠ 术语统一**：本卷正文中所有"灵石"应写作"文石"，"灵气"应写作"文气"。已在 Ch1-15 正文中完成替换，Ch16+ 正文待后续统一。
## 章节列表 (Ch 90 - Ch 100)

### 序幕：风暴前夕
*   **第090章**：[原定] 妖族大军压境，朝堂震动。
*   **第091章**：[原定] 萧千卷在狱中推演战局，获得“兵圣”传承碎片。

### 第一阶段：全面入侵 (生物化学危机)
*   **第092章：地表最强入侵** / *Surface Crisis*
    *   **剧情**：妖族不再是只会肉搏的野兽，使用了类似“生化武器”的毒雾和“变然”狂暴药剂。
    *   **亮点**：萧千卷识别出“病毒”攻击，建议医家进行“生化隔离”。

*   **第093章：城破** / *The Breach*
    *   **剧情**：长安外郭城墙被巨型“攻城兽”（生物坦克）撞破。
    *   **亮点**：常规守城器械失效，绝望蔓延。

### 第二阶段：机械降神 (墨家工业革命)
*   **第094章：谁来守护长安** / *Civilization Upgrade*
    *   **剧情**：萧千卷接管防务，系统发布【文明升级任务】。激活始皇陵沉睡的“兵马俑”（其实是早期的蒸汽动力装甲）。
    *   **系统**：解锁“一级工业文明”科技树。

*   **第095章：机械军团出击** / *Mecha Legion*
    *   **剧情**：墨家弟子驾驶简易外骨骼和“神机弩”（加特林版）登场，构建金属风暴防线。
    *   **分镜**：排枪射击，蒸汽朋克风格的火力覆盖。

*   **第096章：巨神兵决斗** / *Titan Duel*
    *   **剧情**：妖族祭出“祖灵法相”（能量体），萧千卷驾驶试作型决战兵器“非攻·零号机”。
    *   **分镜**：驾驶舱第一视角，HUD数据流，高频振动刀斩杀。

### 第三阶段：百家争鸣 (混合战争)
*   **第097章：百家交响曲** / *Hundred Schools Symphony*
    *   **剧情**：各大学派展现“科学化”战斗力。
    *   **农家**：转基因植物防线（豌豆射手/食人花）。
    *   **医家**：战场急救与生化反制（兴奋剂/解毒剂雾化投放）。
    *   **法家**：区域重力控制（画地为牢·重力井）。

### 第四阶段：终极威慑 (天基武器)
*   **第098章：言出法随·天罚** / *Orbital Strike*
    *   **剧情**：妖皇亲临，常规手段无效。萧千卷利用“言出法随”作为引导信标，呼叫近地轨道打击。
    *   **高潮**：天基武器“定秦星”发射激光束“太阳之光”，精确打击妖皇。

### 尾声：新纪元
*   **第099章：罪己诏** / *The Cyber Dragon*
    *   **剧情**：战后重建。皇帝下罪己诏，承认墨家/杂家地位。国运金龙进化为“五彩数据机械龙”。
    *   **系统**：大梁帝国进入"1.5级文明·文气复苏初级阶段"。

*   **第100章：天下谁人不识君** / *To The Stars*
    *   **剧情**：三个月后，长安赛博化重建。萧千卷拒绝高官厚禄，组建“科考队”前往昆仑。
    *   **结局**：骑乘核能机车“赤兔2000”，目标——昆仑天门（疑似外星人/观察者基地）。

## 关键道具与设定
*   **非攻·零号机**：墨家最高杰作，需文气驱动的灵能机甲。
*   **定秦星**：始皇留下的防御卫星，被误认为是“天罚”。
*   **新·稷下学宫**：改组为“综合理工大学”，分科治学。
