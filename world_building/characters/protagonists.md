# 世界人物志（角色背景与能力）

## 第一梯队：核心主角团（“破局者”）

### 1. 萧千卷
*   **定位**：男主 / 废柴藏书吏 / 活字典
*   **核心能力**：《万经总要》（脑内禁书）、招式【还原】（将篡改招式强制恢复）。
*   **性格**：表面慵懒怕事，实则过目不忘。
*   **伏笔**：身负前朝皇室血脉，随身带着一只会说话的“书虫”。

### 2. 墨染
*   **定位**：女主 / 墨家弃徒 / 机关刺客
*   **核心能力**：《墨子·非攻》、招式【兼爱结界】（物理防御）、操控微型机关傀儡。
*   **特征**：沉默寡言，右臀是机关义肢（刻满铭文）。
*   **目标**：被墨家正统追杀，试图寻找“兼爱”的真正含义。

### 3. 公孙游
*   **定位**：男二 / 纵横家传人 / 嘴炮王者
*   **核心能力**：《鬼谷子》、招式【捭阁之术】（语言诱导幻觉/自相残杀）。
*   **特征**：风流倜傥，毒舌，扇子上写着“以理服人”。智囊担当。

### 4. 蛮山
*   **定位**：护卫 / 兵家遗孤 / 肌肉坦克
*   **核心能力**：《孙子兵法·军争》、招式【侵掠如火】（肉体强化）。
*   **特征**：憨厚耿直，不识字，通过听别人念书获得力量。

## 第二梯队：当世巅峰（“镇世八大家”）

### 1. 儒家·春秋书院（正统维护者）
*   **颜师古（院长）**：立言境巅峰。招式【克己复礼】（领域内强制行礼，限制对手）。
*   **孟浩然（首席大弟子）**：伪君子，早期宿敌。招式【浩然正气】（声量压制）。

### 2. 法家·刑狱司（朝廷鹰犬）
*   **李斯年（司主）**：冷酷无情。招式【画地为牢】（空气墙）、【连坐】（因果伤害）。

### 3. 道家·逍遥观（避世高人）
*   **庄迷蝶（掌门）**：睡不醒的乞丐。招式【逍遥游】（化身大鹏/游鱼，无视碰撞）。

### 4. 兵家·神机营（战争机器）
*   **白起之（大将军）**：杀神转世。招式【风林火山】（士气实体化为战灵）。

## 第三梯队：反派势力（伪经组织“乱神教”）
*   **核心理念**：认为只有篡改经典，释放人性中的恶，才能获得最强力量。
*   **教主·“大祭酒”**：身份成谜。能力【涂改现实】（黑笔抹去规则）。
*   **杀手·“断章”**：专杀读书人。招式【断章取义】（切断招式导致反噬）。
*   **妖女·“颜如玉”**：魅惑众生。招式【书中自有颜如玉】（召唤幻影吸取精气）。

## 第四梯队：特色配角
*   **百晓生**：江湖情报贩子，能力《周易》，擅长预测逃跑，经营“八卦楼”。
*   **钱万贯**：杂家商人，笃信金钱通神，购买孤本雇佣强者。
*   **苏小小**：青楼名妓/隐秘乐家传人，琴声乱心，情报中转站。
*   **老疯子**：前朝守墓人，念叨上古神语（甲骨文），主角隐形导师。
