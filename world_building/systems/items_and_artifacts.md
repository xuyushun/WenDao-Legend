# 修真物品与珍宝图鉴

在大稷王朝与飞升后的真文界，因“文气”取代了“灵气”，所有的物品都与文字、书籍、文房四宝息息相关。

## 一、 丹药体系（方技流出品）
丹药不仅仅是药材的融合，更是对“汉字偏旁部首”的重组。

### 品级划分
*   **凡品**（不论阶，市井可见）：治病强身。
*   **灵品**（对应立言境1-3境）：辅助修炼，激发文气。
*   **玄品**（对应立言境4-5境）：感悟规则，突破瓶颈。
*   **神品**（圣人/飞升级）：起死回生，重塑肉身。

### 常见丹药目录
1.  **墨香丸**（灵品下阶）：
    *   *功效*：恢复少量文气，提神醒脑。
    *   *成分*：墨鱼汁 + “醒”字灰烬。
2.  **洗髓易筋散**（灵品中阶）：
    *   *功效*：排除体内杂质，拓宽书宫。
    *   *成分*：将《易筋经》残篇烧成灰冲服。
3.  **过目不忘丹**（灵品上阶）：
    *   *功效*：短时间内记忆力提升十倍，可强行记下晦涩经文。
    *   *成分*：脑髓 + “记”字结晶。
4.  **浩然丹**（玄品）：
    *   *功效*：儒家专属，大幅增加浩然正气，克制心魔。
    *   *成分*：采集清晨的第一缕紫气 + 百年竹简粉末。
5.  **九转还魂丹**（神品）：
    *   *功效*：只要名字没被地府判官笔勾掉，就能救活。
    *   *成分*：凤凰涅槃火 + “生”字本源。

## 二、 武器与文宝（兵器谱）
修士的武器多为文具衍生物，或刻满铭文的常规兵器。

### 武器类
1.  **判官笔**（通用）：
    *   *特点*：只有笔尖，没有笔毛，用陨铁打造。点穴、写符专用。
2.  **量天尺**（儒家）：
    *   *特点*：一把厚重的戒尺。打人专打手心，具有“教化”压制效果，让敌人产生羞愧感。
3.  **君子剑**（儒/剑修）：
    *   *特点*：剑身刻有《论语》，平时藏在剑鞘（竹筒）里，出剑必见血。
4.  **非攻盾**（墨家）：
    *   *特点*：一面巨大的青铜盾牌，平时折叠在手臂上，展开后可抵挡攻城弩。
5.  **逍遥扇**（道/纵横）：
    *   *特点*：扇面是空白的，战斗时可以随机显现出不同的法术图案。
6.  **杀猪刀**（屠夫/小说家）：
    *   *特点*：虽然粗俗，但上面刻着“庖丁解牛”四字，无视护甲。

### 防具类
1.  **浩然巾**（头饰）：防止精神攻击和读心术。
2.  **流云水袖**（道家法衣）：袖里乾坤，可以收纳物品或卷走敌人的兵器。
3.  **软猥甲**（黄蓉同款）：上面密密麻麻写满了微雕的“刺”字，谁打谁手疼。
4.  **步云履**（鞋）：鞋底刻着“缩地成寸”，增加移动速度。

## 三、 天材地宝（文材）
用于打造书宫或强化本命字灵的珍稀材料。

1.  **龙骨拓片**：
    *   *来源*：上古遗迹。
    *   *用途*：参悟最古老的甲骨文力量。
2.  **在此一游石**：
    *   *来源*：古迹名胜。
    *   *用途*：蕴含前人留下的精神印记，可用来强化武器的“名气”。
3.  **文心雕龙血**：
    *   *来源*：斩杀亚龙种获得。
    *   *用途*：墨水原材料，写出的字自带龙威。
4.  **七窍玲珑墨**：
    *   *来源*：传说中的神石研磨而成。
    *   *用途*：用此墨写出的字，会自己思考变通。

## 四、 秘宝与奇物（功能性道具）
1.  **无字天书**（概念级）：
    *   *效果*：平时是空白的，遇到危机时会浮现出唯一的解局之法。
2.  **免死金牌**（法家圣物）：
    *   *效果*：帝王御赐，刻有“死刑豁免”律令。可抵挡一次必死法则攻击。
3.  **月光宝盒**（时空类）：
    *   *效果*：大话西游同款。对着月亮喊“波若波罗密”，可回溯时光（极不稳定）。
4.  **神笔马良（仿）**：
    *   *效果*：一次性消耗品。画什么出什么，但画出的东西只能存在一刻钟。

## 五、 副本与古迹（探险地）

### 初级副本（1-2境）
1.  **翰林书院废墟**：
    *   *怪物*：书虫、墨鬼、发疯的书生冤魂。
    *   *掉落*：残破的经书、旧毛笔、低级丹药。
2.  **兰亭集序秘境**（流水曲水）：
    *   *玩法*：在流动的溪水中抢夺漂流的酒杯，喝下酒会获得Buff，也可能中毒。
    *   *掉落*：《兰亭集序》摹本碎片。

### 中级副本（3-4境）
1.  **桃花源记**：
    *   *设定*：一个无限循环的时空迷宫。只有看破“不足为外人道也”的真谛才能走出。
    *   *掉落*：延寿桃、隐世秘籍。
2.  **赤壁战场**：
    *   *设定*：重现火烧赤壁。玩家需选择阵营（曹/孙刘），在大火中生存。
    *   *掉落*：东风符、连环锁战船图纸。

### 高级/禁地副本（5境+）
1.  **秦始皇陵（兵马俑坑）**：
    *   *设定*：所有的兵马俑都是活的战阵。深处藏着祖龙焚书的灰烬。
    *   *掉落*：法家至高律令、祖龙魂魄。
2.  **山海经·大荒界**：
    *   *设定*：完全由异兽统治的世界。
    *   *掉落*：神兽坐骑、补天石。

## 六、 功法宝典（经史子集）
在这个世界，功法就是书籍。

1.  **【经部】（正统心法）**：
    *   《易经》（预测/万能）、《诗经》（音律/召唤）、《春秋》（因果律）。
2.  **【史部】（战斗技法）**：
    *   《战国策》（纵横术）、《史记》（刺客列传/游侠列传）。
3.  **【子部】（百家秘术）**：
    *   《墨子》（机关术）、《庄子》（身法/幻术）、《韩非子》（控制/刑法）。
4.  **【集部】（爆发大招）**：
    *   《唐诗三百首》（战诗合集）、《楚辞》（大型祭祀咒术）。
